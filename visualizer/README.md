# Visualizer

This visualizer is the frontend for the Autonomous Delivery Navigation Agent simulator. It renders preset scenarios and custom maps in 3D, runs Planner-Guided DQN replay, and supports direct 3D map editing.

## Features

- Planner-Guided DQN replay for preset scenarios and custom maps
- fullscreen scene mode with collapsible side panels
- direct 3D tile editing plus optional 2D editor
- smooth time-based robot playback
- dynamic obstacle synchronization
- planner route and executed route visualization

## Development

```bash
npm install
npm run dev
```
